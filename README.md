# Smart Lender - Loan Approval Prediction

## Project Overview

Smart Lender is a Machine Learning web application that predicts whether a loan application is likely to be approved based on applicant details.

The project helps banks and financial institutions automate loan approval decisions using historical applicant data.

---

## Features

- Data Exploration (EDA)
- Data Preprocessing
- Missing Value Handling
- Label Encoding
- Feature Scaling
- Model Training
- Flask Web Application
- Loan Approval Prediction

---

## Machine Learning Models

- Decision Tree
- Random Forest
- K-Nearest Neighbors
- Gradient Boosting (Best Model)

## How to Run

Clone the repository

```bash
git clone https://github.com/Komal38/Smart_Lender_Loan_Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Flask

```bash
python app/app.py
```

Open browser

```
http://127.0.0.1:5000
```
